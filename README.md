# ระบบลงเวลางานและคำนวณเงินเดือนผ่าน LINE OA
(Work Attendance and Payroll System via LINE OA)

## 1. ชื่อโครงการ
**ระบบลงเวลางานและคำนวณเงินเดือนผ่าน LINE OA**

## 2. ขอบเขตระบบ

### 2.1 ขอบเขตด้านเนื้อหา
ระบบนี้ออกแบบมาสำหรับใช้งานภายใน **บริษัทเดียว (Single Organization)** โดยแบ่งผู้ใช้งานออกเป็น 2 บทบาทหลัก:

#### 1) เสมียน (User / Operator)
เป็นผู้ใช้งานหลัก (Power User) ที่ทำหน้าที่จัดการข้อมูลเกือบทั้งหมดในระบบ
*   **จัดการข้อมูลพนักงาน:** เพิ่ม, แก้ไข, ลบข้อมูลพนักงาน และกำหนดอัตราเงินเดือน (รายวัน, รายเดือน, OT)
*   **บันทึกเวลาและงาน:** ลงเวลาเข้า-ออกงานแทนพนักงาน, บันทึกรายละเอียดงานรายวัน และแก้ไขข้อมูลย้อนหลังได้
*   **จัดการงวดเงินเดือน (Payroll Period):** กำหนดช่วงวันที่ (เช่น วันที่ 1–15 หรือ 16–สิ้นเดือน) และทำการล็อกงวดเมื่อสรุปข้อมูลเสร็จสิ้น
*   **คำนวณเงินเดือน:** ประมวลผลจากจำนวนวันทำงาน ชั่วโมงทำงาน และ OT เพื่อสรุปยอดรายพนักงานในแต่ละงวด
*   **ดูรายงาน:** ตรวจสอบความถูกต้องและเรียกดูข้อมูลสรุปต่างๆ

#### 2) เจ้านาย / ผู้บริหาร (Admin / Owner)
เน้นการติดตามและตรวจสอบข้อมูล (Read-only / Overview)
*   ดูรายงานเวลาทำงานของพนักงาน
*   ดูสรุปรายละเอียดงานรายวันและรายเดือน
*   ดูสรุปยอดเงินเดือนภาพรวม
*   ตรวจสอบความถูกต้องของข้อมูลที่เสมียนบันทึก

### ลักษณะข้อมูลหลักในระบบ
*   **พนักงาน:** ชื่อ-นามสกุล, ตำแหน่ง, อัตราค่าจ้าง,
*   **เวลาทำงาน:** บันทึกเวลาเข้า-ออก และวันที่
*   **รายละเอียดงาน:** ข้อความบรรยายงานที่ทำในแต่ละวัน
*   **งวดเงินเดือน:** รอบการจ่ายเงินและสถานะการปิดงวด
*   **ผลคำนวณเงินเดือน:** ยอดเงินสุทธิและรายละเอียดการคำนวณ

---

### 2.2 ขอบเขตด้านระบบ
#### แพลตฟอร์ม (Platform)
*   ใช้ **LINE Official Account (LINE OA)** เป็นช่องทางหลักในการใช้งาน (Frontend UI)

#### ฟีเจอร์หลักบน LINE (Rich Menu)
1.  **จัดการพนักงาน:** เมนูสำหรับบริหารจัดการข้อมูลพื้นฐานพนักงาน
2.  **สร้างงวด:** เมนูสำหรับกำหนดรอบการตัดวิกเงินเดือน
3.  **ลงเวลางานและบันทึกงาน:** เมนูสำหรับบันทึกเวลาเข้า-ออกและรายละเอียดงาน
4.  **คำนวณเงินเดือน:** เมนูสำหรับสั่งประมวลผลยอดเงินเดือน
5.  **ดูรายงาน:** เมนูสำหรับเรียกดูสรุปผลในรูปแบบต่างๆ
